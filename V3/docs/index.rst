Overview
--------

AutoPharm Core is the core library for the PharmaControl V3 autonomous control system. It provides the modules for:

* Online learning and adaptation
* Reinforcement learning for policy optimization
* Explainable AI for transparency and trust

The system is designed to be modular and extensible, with a microservice architecture that allows for easy deployment and scaling.
